VANET P2P Implementation Documentation
=====================================

Modules Developed (Post Phase-2)
-------------------------------

1. Vehicle Node Module (src/vehicle_node.py)
------------------------------------------
Functionality: Core implementation of individual vehicle nodes in the P2P network
Implementation Details:
- Class: VehicleNode
  * Attributes:
    - node_id: Unique identifier for each vehicle
    - location: (latitude, longitude) tracking
    - speed, direction: Movement parameters
    - connections: List of peer connections
    - trust_score: Node reliability metric
  * Features:
    - P2P communication methods using libp2p
    - Status update mechanisms
    - Location and speed tracking
    - Connection management

2. Clustering Algorithm (src/clustering.py)
----------------------------------------
Functionality: Manages dynamic cluster formation and node role assignment
Implementation Details:
- Class: ClusterManager
  * Clustering Metrics:
    - Relative mobility calculation
    - Direction similarity metrics
    - Signal strength assessment
  * Node Weight Calculation:
    - Connectivity degree (25%)
    - Relative mobility (25%)
    - Signal strength (20%)
    - Battery level (15%)
    - Historical stability (15%)
  * Features:
    - Dynamic cluster maintenance
    - Boundary node detection
    - Cluster head selection
    - Member management

3. Consensus Mechanism (src/consensus.py)
---------------------------------------
Functionality: Implements PoA consensus for relay node selection
Implementation Details:
- Class: ConsensusManager
  * Components:
    - Network graph management
    - Node centrality calculations
    - Authority score tracking
    - Relay node selection algorithm
  * Features:
    - Performance-based authority updates
    - Consensus validation
    - Network state management

4. Network Manager (src/network_manager.py)
----------------------------------------
Functionality: Handles network stability and fault tolerance
Implementation Details:
- Class: NetworkManager
  * Core Functions:
    - Active node tracking
    - Relay node management
    - Health monitoring
    - Failure detection
  * Features:
    - Network statistics collection
    - Performance monitoring
    - State synchronization

5. Simulation Integration (src/simulation_bridge.py, src/custom_vanet_appl.py)
--------------------------------------------------------------------------
Functionality: Integrates P2P VANET with OMNeT++/SUMO/Veins
Implementation Details:
- Components:
  * SUMO-OMNeT++ synchronization
  * Vehicle state updates
  * Message handling
  * Real-time visualization
- Features:
  * Traffic simulation integration
  * Network simulation
  * Visualization tools
